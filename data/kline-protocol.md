# 📈 板块日K走势实时量化分析规范（Sector K-line Quant Protocol）

> 本规范定义尾盘决策前的**板块级日K量化分析框架**：用 5 大维度把"板块走势"转化为可计算的信号，
> 重点回答四个问题：
> ① **主力是否给出特殊信号？** ② **是否止跌 / 见顶？** ③ **主力在洗盘建仓 还是 抢筹出货？** ④ **收盘价有无特殊信号？**
> 目标：让尾盘决策从"看涨跌"升级为"读主力意图 + 辨形态信号 + 识收盘价暗号"。

---

## 一、五维分析框架

| 维度 | 回答的问题 | 输出 |
|------|-----------|------|
| 1. 趋势定位 | 现在是多头/空头/震荡？ | 趋势评分 0-100 + 均线排列 |
| 2. 量价关系 | 价量是否配合？有无背离？ | 量价健康度 0-100 |
| 3. 形态信号 | 止跌 or 见顶？ | 止跌✓ / 见顶⚠ / 无 |
| 4. 主力行为 | 洗盘/建仓/抢筹/出货？ | 主力意图判定 |
| 5. 收盘价暗号 | 收盘有无特殊信号？ | 信号标签列表 |

> 所有维度评分 0-100：0=极空，50=中性，100=极多。综合加权得**板块多空倾向**。

---

## 二、维度1：趋势定位（Trend）

**均线排列判定**：
```
强多：MA5>MA10>MA20>MA60 且 收盘价在所有均线之上
偏多：收盘价在 MA20 之上，均线多头但未完全发散
震荡：收盘价在 MA5~MA60 之间纠缠，无明确方向
偏空：收盘价在 MA20 之下，均线空头但未完全发散
强空：MA5<MA10<MA20<MA60 且 收盘价在所有均线之下
```

**趋势评分映射**：
```
强多 → 85-100   偏多 → 65-84   震荡 → 40-60
偏空 → 16-39    强空 → 0-15
```
辅助：MA20 斜率向上+10、MA60 斜率向上+5（向下则减）；价格距 MA20 上方每 2% +3，下方每 2% -3（封顶±10）。

## 三、维度2：量价关系（Volume-Price）

| 组合 | 含义 | 量价健康度 | 主力暗示 |
|------|------|-----------|---------|
| 价涨量增 | 健康上涨 | 80-100 | 抢筹/持续做多 |
| 价跌量增 | 恐慌抛售 | 0-30 | 出货/多杀多 |
| 价涨量缩 | 无量上涨 | 40-60 | 惜售但动能不足 |
| 价跌量缩 | 抛压减轻 | 50-70 | 可能止跌/洗盘末端 |
| 价平量平 | 僵持 | 50 | 观望 |

**背离检测**：
- **顶背离**：价格创新高，但成交量 / 主力净流入未创新高 → 见顶预警
- **底背离**：价格创新低，但成交量萎缩至地量 / 主力净流出收窄 → 止跌预警

## 四、维度3：形态信号（止跌 vs 见顶）

### 🟢 止跌信号（底部，偏多）
| 信号 | 形态条件 |
|------|---------|
| 锤头/长下影 | 下影线 ≥ 2×实体，且出现在下跌后 |
| 十字星 | 开盘≈收盘，实体极小，变盘意味 |
| 缩量止跌 | 连续下跌后成交量萎缩至近期地量 |
| 阳包阴(看涨吞没) | 今日阳线实体完全包裹昨日阴线 |
| 探底回升 | 盘中大跌（下影长）但收盘拉回 |
| 底背离 | 价新低而量/资金未新低 |

### 🔴 见顶信号（顶部，偏空）
| 信号 | 形态条件 |
|------|---------|
| 射击之星/长上影 | 上影线 ≥ 2×实体，出现在上涨后 |
| 高位放量滞涨 | 大涨但收盘回落，留长上影，成交量放大 |
| 阴包阳(看跌吞没) | 今日阴线实体完全包裹昨日阳线 |
| 高位十字星 | 连续上涨后出现十字星，变盘 |
| 顶背离 | 价新高而量/资金背离 |
| 利好不涨 | 出利好却高开低走、放量长上影 |

> 止跌+底背离+缩量 = 强止跌（洗盘末端）；见顶+顶背离+放量 = 强见顶（出货）。

## 五、维度4：主力行为识别（核心）

| 行为 | 量价特征 | K线特征 | 盘口/资金特征 | 操作含义 |
|------|---------|--------|-------------|---------|
| 🟢 **建仓** | 低位**温和放量** | 连续小阳线，波动小 | 主力净流入温和持续，散户无关注 | 逢低布局区，不急于追 |
| 🟢 **抢筹** | 突然**放巨量** | 大阳线突破平台/均线 | 主力净流入陡升，OBV急拉 | 趋势启动，可顺势跟进 |
| 🟡 **洗盘** | **缩量**震荡下行 | 不破关键支撑(MA20/前低)，尾盘常拉回 | 主力净流入小幅为正或持平，散户恐慌出逃 | 持有/低吸，勿被震出 |
| 🔴 **出货** | 高位**放量** | 长上影/阴线，冲高回落 | 主力净流出放大，对倒放量 | 减仓/止盈，不接飞刀 |

**判别口诀**：
```
不破支撑 + 缩量 + 尾盘拉回 = 洗盘（拿住）
低位小阳 + 温和放量 + 时间长 = 建仓（布局）
巨量长阳 + 突破 = 抢筹（跟进）
高位长上影 + 放量 + 净流出 = 出货（撤退）
```

## 六、维度5：收盘价特殊信号（Closing Price Special）

| 信号 | 条件 | 含义 |
|------|------|------|
| 站上MA5/10/20/60 | 收盘突破某均线 | 对应周期转强（突破信号） |
| 跌破MA5/10/20/60 | 收盘跌破某均线 | 对应周期转弱（破位信号） |
| 光头阳线 | 收盘=当日最高 | 极强，次日易高开 |
| 光脚阴线 | 收盘=当日最低 | 极弱，次日易低开 |
| 十字星 | 收盘≈开盘 | 变盘临界点 |
| 大阳线 | 实体 ≥ +3% | 强势确认 |
| 大阴线 | 实体 ≤ -3% | 弱势确认 |
| 向上跳空缺口 | 收盘>昨最高 | 突破性缺口（强） |
| 向下跳空缺口 | 收盘<昨最低 | 破位性缺口（弱） |
| 地量 | 成交量创近期新低 | 情绪极致（地量见地价） |
| 天量 | 成交量创近期新高 | 分歧极致（天量见天价） |

## 八、与尾盘决策/情绪因子的关系

- **K线量化（板块面）**：告诉我们"这个板块现在处于什么技术阶段、主力在干嘛"
- **情绪因子（市场面）**：告诉我们"全市场现在有多恐慌/贪婪"
- **二者叠加**：
  - 板块止跌 + 市场恐惧 → 高赔率左侧买点（洗盘建仓区）
  - 板块见顶 + 市场贪婪 → 高确定止盈点（抢筹出货区）
  - 板块抢筹 + 市场中性 → 顺势跟进但控仓
  - 板块出货 + 市场恐慌 → 果断减仓，不抄底

- 严禁用 T3 自媒体"主力已建仓"等表述直接作为判定依据
