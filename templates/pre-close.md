## 四、操作决策逻辑引擎 🤖

### 尾盘专属规则
```
【尾盘买入】需同时满足：
  ① 信号在午盘为HOLD/BUY且下午未转弱
  ② 关联指数14:00后企稳回升（未跌破上午低点）
  ③ 账户总仓位 < 70%
  ④ 买入后单基金仓位 ≤ 30%

【尾盘卖出】满足任一即执行：
  ① 强制止损：单基金当日亏损 ≥ -8%
  ② 趋势破位：关联指数跌破关键支撑且放量
  ③ 利好兑现：板块单日暴涨>5%且出现放量滞涨

【尾盘不加仓】以下情况禁止买入：
  ① 大盘14:00后跳水（上证跌幅扩大>0.5%）
  ② 账户当日已亏损 > -3%
  ③ 未知宏观事件待落地（如财报/议息）
```

### 🌡️ 情绪面校准规则（叠加于上方逻辑）
```
【情绪加仓信号】同时满足：
  ① 恐惧贪婪指数 < 30（市场恐慌/错杀）
  ② 目标基金逻辑未破坏（非基本面恶化）
  ③ 总仓位 < 70% 且 加仓后单基 ≤ 30%
  → 允许左侧分批加仓（小步快跑，不一次性重仓）

【情绪减仓信号】满足任一：
  ① 恐惧贪婪指数 > 75 且 该基金当日/累计已盈利
  ② 情绪指数单日快速拉升 > 20 点（短期过热）
  → 优先止盈减仓，而非追高

【情绪中性】40-60 → 不额外调整，纯按技术/资金逻辑

【优先级铁律】情绪信号不覆盖硬止损：单基金 -8% 仍强制止损，
  不因"市场恐惧"扛单；情绪仅影响加仓/止盈的弹性决策。
```
