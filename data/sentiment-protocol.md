# 🌡️ 市场情绪因子分析规范（Sentiment Factor Protocol）

> 本规范定义「市场情绪投资」框架：通过 8 个可量化情绪因子，合成 **恐惧贪婪指数（0-100）**，
> 作为尾盘决策的**第四维校准因子**（技术面 / 基本面 / 资金面 / 情绪面）。
> 目标：把"市场情绪"从模糊的直觉，变成可计算、可回溯、可问责的量化信号。

---

## 一、核心思想：情绪面为何重要

> **价格围绕价值波动，但短期由情绪定价。**

- 技术面告诉你"现在在哪"，基本面告诉你"该值多少"，资金面告诉你"钱往哪去"
- **情绪面告诉你"市场现在有多疯 / 有多慌"** —— 这是逆向投资与止盈时机的核心锚
- 散户亏钱的主因之一：**在贪婪时追高、在恐惧时割肉**。情绪因子强制逆向校准。

---

## 二、8 大情绪因子（加权合成 0-100）

每个因子独立评分 **0-100**（0=极度恐惧，100=极度贪婪），按权重合成综合指数。

| # | 因子 | 数据来源 | 锚定逻辑 | 权重 |
|---|------|---------|---------|------|
| 1 | **市场广度** | T2 东财 | 涨跌家数比（涨/跌），>1 偏贪婪 | 15% |
| 2 | **涨跌停家数** | T2 | 涨停多=贪婪，跌停多=恐惧 | 10% |
| 3 | **主力资金净流入** | T2 | 全市场/半导体主力净流入额 | 15% |
| 4 | **北向资金净买入** | T1/T2 | 陆股通当日净买入（亿元） | 12% |
| 5 | **融资余额变化** | T2 | 两融余额环比，上升=杠杆追涨 | 8% |
| 6 | **量能亢奋度** | T2 | 成交额 vs 近20日均值，异常放量=极端 | 10% |
| 7 | **VIX 恐慌指数** | T2 | 美股 VIX，>30=恐惧，<15=贪婪 | 15% |
| 8 | **股债风险溢价 ERP** | T2 | 沪深300 PE倒数 - 10Y国债收益率，高=权益便宜 | 15% |

> 权重合计 100%。VIX 与 ERP 给高权重，因其是**跨市场、低频噪声、逆向性强**的锚。

### 单因子评分映射（示例：市场广度）
```
涨/跌家数比    ≥ 2.0  → 90-100（极度贪婪）
               1.5-2.0 → 75-89
               1.0-1.5 → 60-74
               0.7-1.0 → 45-59（中性）
               0.5-0.7 → 30-44
               0.3-0.5 → 15-29
               < 0.3   → 0-14（极度恐惧）
```

> 各因子具体映射阈值见下方「评分卡」模板，可由 AI 按当日数据在报告中实时填充。

## 三、恐惧贪婪指数分级与交易含义

| 区间 | 情绪 | 尾盘决策含义 |
|------|------|------------|
| **0-20** | 极度恐惧 😱 | **左侧买入区**：优质持仓被错杀可分批加仓（遵守总仓≤70%、单基≤30%） |
| **20-40** | 恐惧 😐 | 中性偏多：可小仓试探，不追高 |
| **40-60** | 中性 😶 | **按技术/基本面逻辑决策**，情绪不额外干预 |
| **60-80** | 贪婪 🤤 | 谨慎：持仓已盈利优先减仓/止盈 |
| **80-100** | 极度贪婪 🔥 | **减仓/清仓预警**：即使技术面强也要警惕，分批止盈 |

> 逆向原则：**别人恐惧我贪婪，别人贪婪我恐惧**。情绪极端时，与情绪反向操作（需逻辑支撑）。

## 四、情绪因子如何接入尾盘决策

尾盘决策 = 技术信号 + 基本面 + 资金流 + **情绪校准（本规范）**

### 情绪校准规则（叠加在原逻辑引擎之上）

```
【情绪加仓信号】同时满足：
  ① 恐惧贪婪指数 < 30（市场恐慌）
  ② 目标基金逻辑未破坏（非基本面恶化）
  ③ 总仓位 < 70% 且 加仓后单基 ≤ 30%
  → 允许左侧分批加仓（小步快跑，不一次性重仓）

【情绪减仓信号】满足任一：
  ① 恐惧贪婪指数 > 75 且 该基金当日/累计已盈利
  ② 情绪从历史低位快速拉升 >20 点（短期过热）
  → 优先止盈减仓，而非追高

【情绪中性】40-60 → 不额外调整，纯按技术/资金逻辑
```

### 情绪与硬止损的优先级
- **情绪信号不改变硬规则**：单基金 -8% 仍强制止损，不因"市场恐惧"而扛单
- 情绪只影响「是否加仓 / 是否止盈」的弹性决策，不覆盖风控底线

## 六、数据可信度

- 因子 #1/#2/#3/#5/#6：T2（东财/同花顺），需双源验证，标注 `⚠ 待验证`
- 因子 #4：T1（交易所/港交所官方陆股通）
- 因子 #7：T2（CBOE VIX 官方）
- 因子 #8：T2（计算值，国债收益率+PE）
- **严禁用 T3 自媒体情绪直接进指数**；舆情热度仅作辅助，且须标注 `⚠ 观点来源`
- 若某因子数据缺失，按中性（50分）计入并标注 `[数据缺失]`，权重不变

## 七、记录与回溯

每日尾盘情绪指数写入 `data/sentiment-log.json`，结构同 `prediction-log.json`：
```json
{
  "date": "2026-07-08",
  "fear_greed_index": 42,
  "level": "中性",
  "factors": { "breadth": 48, "limit_up_down": 40, "main_flow": 35, "northbound": 55, "margin": 50, "volume": 45, "vix": 30, "erp": 60 },
  "sentiment_action": "中性，按技术逻辑决策",
  "note": "隔夜美存储暴跌压制，VIX偏高，但A股广度尚可"
}
```
> 回溯用途：情绪极端日的历史表现、情绪指标对次日涨跌的预测力（类似 prediction-log 命中率追踪）。
